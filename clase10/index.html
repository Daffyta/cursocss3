Propiedad animation

Permiten animar la transición entre un estilo CSS y otro. Las animaciones constan de dos componentes: un estilo que describe la animación y un conjunto de fotogramas que indican su estado inicial y final, así como posibles puntos intermedios en la misma.

Las animaciones CSS tienen tres ventajas principales sobre las técnicas tradicionales de animación basada en scripts:

  1. Es muy fácil crear animaciones sencillas, incluso sin tener conocimientos de Javascript.
  2. La animación se muestra correctamente, incluso en equipos poco potentes.
  3. Al ser el navegador controle la secuencia de la animación, permitimos que optimice el rendimiento y eficiencia de la misma

Configurando la animación

Para realizar una animación se requiere del uso de dos elementos:

animationNombre de la porpiedad CSS3
@keyframesPermite declarar los fotogramas claves

Propiedad animation

Esta propiedad consta de varias sub-propiedades. Con ellas podemos configurar el ritmo, la duración, el nombre, la función de tiempo y otros detalles sobre la secuencia de la animación.

Sub-propiedades de animación son:

animation-delayTiempo de retardo en el comienzo de animación.
animation-directionIndica el comprtamiento de la animación al inicio y al final.
animation-durationCantidad de tiempo que tardará en completar su ciclo (duración).
animation-iteration-countEl número de veces que se repite.

Sub-propiedades de animación son:

animation-nameEl nombre de la animación.
animation-fill-modeQué estilos se aplicarán al elemento cuando la animación no se este ejecuntando
animation-play-statePermite pausar y reanudar la animación.
animation-timing-functionIndica el ritmo de la animación.

Sub-propiedad [animation-direction]

Consta de valores predefinidos que alteran la dirección de la animación:

normal: La animación se reinicia comenzando desde el principio.

alternate: La animación, al terminar un ciclo, invierte su dirección. Es decir, los pasos de la animación se ejecutan al revés.

reverse:Cada ciclo de la animación se reproduce al revés

alternate-reverse:La animación se posiciona en el estado final, comienza a reproducirse al reves y, cuando llega al inicio vuelve a reproducirse de forma normal hasta llegar al final.

Sub-propiedad [animation-timing-function]

Consta de valores predefinidos que alteran la dirección de la animación:

linearLa animación se realiza de manera uniforme.
easeAcelera al inicio se retarda y se acelera al final.
ease-in Se retarda al inicio, pero lo repone al final.
ease-out Acelera al inicio pero se retarda al final.
ease-in-out Se retarda al inicio, acelera y luego se retarda al final.
steps(-número-, end) divide la animación en un número de pasos

Ejemplo Propiedad animacón:

						
h1 {
  animation-duration: 3s;
  animation-name: h1animacion;
  animation-iteration-count: infinite;
  animation-direction: normal;
}

Con esto indicamos parametros de la animacón que deberán ejecutarse. Pero aún nos hace falta declarar la regla @keyframe para ver algun resultado.

@keyframes

Permite controlar los pasos en una secuencia de animación CSS mediante el establecimiento de puntos de trayectoria (@keyframes) en la secuencia de animación.

Para utilizar keyframes, se crea una regla @keyframes con el mismo nombre declarado en la propiedad animation-name para que coincida con la animación de keyframe.

@keyframes

Para obtener una regla @keyframe válida, debe incluir reglas para al menos los tiempos 0% (from) y 100% (to) que son los estados inicial y final de la animación. Si ambos desplazamientos de tiempo no se especifican, la declaración @keyframe es inválida y no se puede utilizar para la animación.

						
@keyframes -nombre-animacion- {
      0%{propiedad: valor}
      100%{propiedad: valor}
      }

Ejemplo regla @keyframes

						
@-webkit-keyframes h1animacion /* Safari and Chrome */
{
0%   {font-size: 10px;}
100% {font-size: 24px;}
}

@keyframes h1animacion
{
0%   {font-size: 10px;}
100% {font-size: 24px;}
}

Ahora añadiremos el código a nuestra práctica:

						
h1 {

	-webkit-animation-duration: 3s;
  	-webkit-animation-name: h1animacion;
  	-webkit-animation-iteration-count: 1;
  	-webkit-animation-direction: normal;

  	animation-duration: 3s;
  	animation-name: h1animacion;
  	animation-iteration-count: 1;
  	animation-direction: normal;
}


@-webkit-keyframes h1animacion /* Safari and Chrome */
{
0%   {font-size: 10px;}
100% {font-size: 24px;}
}

@keyframes h1animacion
{
0%   {font-size: 10px;}
100% {font-size: 24px;}
}

Ahora con ambas reglas la animación se ejecuta, si el navegador tiene soporte.

Ahora cambiaremos los valores de porcentaje (0% y 100%) en @keyframe por from y to 

						

@-webkit-keyframes h1animacion /* Safari and Chrome */
{
from   {font-size: 10px;}
to {font-size: 24px;}
}

@keyframes h1animacion
{
from   {font-size: 10px;}
to {font-size: 24px;}
}

Ambas formas son correctas.

Trabajar con porcentajes permite agregar más pasos en la secuencia de animacónfrom y to 

						

@-webkit-keyframes h1animacion /* Safari and Chrome */
{
 0%{font-size: 10px;}
 30%{color: green;}
 60%{color: blue;}
100% {font-size: 24px; color: #fff;}
}

@keyframes h1animacion 
{
 0%{font-size: 10px;}
 30%{color: green;}
 60%{color: blue;}
100% {font-size: 24px; color: #fff;}
}
