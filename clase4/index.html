Propiedades de Dimensión

Se utilizan para definir el ancho y alto de un elemento

height: altura del elemento (px, em, pt, %, auto)
width:ancho del elemento (px, em, pt, %, auto)
max-width: Ancho máximo
min-width:Ancho mínimo
max-heightAltura máxima
min-height;Atura mínima

Propiedades de Fuente

Se utilizan para añadir atributos especiales a la fuente.

font-family:Fuente específica (Arial, Times, Verdana, Georgia, Tahoma) Familia (serif, sans-serif, monospace, cursive)
font-style:normal, italic, oblique
font-weight: lighter, normal, bold(er). Valor numérico (100, 200, 300, 400, 500, 600, 700, 800, 900)
font-size:Tamaño en puntos (pt), cm, %, ems
font-variant:normal, small-caps
font:font: bold 1.2em arial;

PropiedadesTextos y párrafos

Se utilizan para añadir atributos especiales a los párrafos y textos.

color: red, green / hexadecimal= #ff0000 / RGB = rgb(112, 122, 122)
line-height:ems, px, pt (define el espacio entre lineas)
text-align left, center, right o justify
text-decorationunderline(subrayado), line-through(tachado), overline (línea sobre el texto) o none
text-transformuppercase (mayúscula), lowercase (minúscula) o capitalize (primea letra en mayúscula)
word-spacingInserta espacio entre las palabras (px, em, pt)

Propiedades de Fondo

Se utilizan para modificar el fondo de un elemento.

background-color: red, green / hexadecimal= #ff0000 / RGB = rgb(112, 122, 122)
background-image:url("//url")
background-repeat repeat, repeat-x, repeat-y, no-repeat
background-attachmentscroll, fixed
background-positiontop, middle, bottom, left, center o right

La propiedad background, se puede escribir en una sola declaración:

						
selector{
	background: url("../carpeta/nombre-imagen.jpg") repeat-x scroll rigt top transparent;
}

Propiedades de Margen

Se utilizan para modificar la separación entre elementos

margin-top: Valor del margen superior (px, em, pt)
margin-right:Valor del margen derecho (px, em, pt)
margin-bottom: Valor del margen inferior (px, em, pt)
margin-left:Valor del margen izquierdo (px, em, pt)
margin:5px 0.5em 2pt 0; (arriba, derecha, abajo, izquierda)
margin:auto;el navegador calcula el valor de los margenes

Propiedades de Bordes

Se utilizan para modificar el borde de un elemento

border-width: Grosor del borde (px, em, pt)
border-color:Color del borde (rgb), hexadecimal, predefinido
border-style: solid (sólido), dashed (con trazos), dotted (con puntos), double (dos líneas) o ridge (tridimensional)
border:border: 1px dotted green;
border[-top -left -bottom -right]:border-left: 1px dotted green;

Propiedades de Padding

Se utilizan para modificar el espacio interno de un elemento respecto a su contenido

padding-top: Relleno entre el elemento y el borde superior (px, em, pt)
padding-right:Relleno entre el elemento y el borde derecho
padding-bottom: Relleno entre el elemento y el borde inferior
padding-left:Relleno entre el elemento y el borde izquierdo
padding:padding: 5px (agrega el valor de padding a todo el elemento)
padding:padding: 5px 4pt 2em 0; (asigna cada valos a cada lado: arriba, derecha, abajo izquierda)

Propiedades de Listas

Modifica el estilo de los elementos de una lista

list-style-type:decimal, upper-roman, lower-latin, disc, circle, square o none
list-style-position: inside o outside
list-style-image:list-style-image: url(image.png);
list-style:list-style: square outside none;
