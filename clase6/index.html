Propiedades Float

Se utiliza para posicionar un elemento moviéndola todo lo posible a la izquierda o derecha de su posición original. Los tres valores de esta propiedad se interpretan de la siguiente manera:

left:el elemento se desplaza todo lo posible a la izquierda de la posición en la que se encontraba.
right:el elemento se desplaza todo lo posible a la derecha de la posición en la que se encontraba.
none: el elemento no se desplaza respecto de su posición original. Además de ser el valor por defecto de esta propiedad.

La principal característica del posicionamiento float es que el resto de elementos de la página se adaptan para fluir alrededor de la caja flotante.

    Propiedad Transition

    Proporciona una forma de animar los cambios de las propiedades CSS, en lugar de que los cambios surtan efecto de manera instantánea.

    
selector {
    transition:[Propiedad CSS] [Tiempo] [Función de Tiempo] [Retardo];
}
				
				
    
selector {
    transition:[Propiedad CSS] [Tiempo] [Función de Tiempo] [Retardo];
}

    [Propiedad CSS]= Refiere a la propiedad CSS que será animada.

    Se puede colocar cada porpiedad o se puede abreviar con el valor: all

    Propiedades que se pueden transicionar

    background-colorcolor
    background-positionUbicación (porcentaje, longitud)
    background-size Tamaño de background
    border-color Color del borde
    border-radius Porcentaje o medida

    Propiedades que se pueden transicionar

    bottommedida (porcentaje, longitud)
    topmedida (porcentaje, longitud)
    box-shadow Sombra
    color Color
    font-size Porcentaje o medida

    Propiedades que se pueden transicionar

    font-weightvalor/palabras clave (excluyendo bolder, lighter)
    leftmedida (porcentaje, longitud)
    right medida (porcentaje, longitud)
    line-height (porcentaje, longitud)
    opacity Valor

    Propiedades que se pueden transicionar

    visibilityvisibilidad
    z-indexnúmero entero
    vertical-align porcentaje, longitud y palabras claves
    line-height (porcentaje, longitud)

    
selector {
    transition:[Propiedad CSS] [Tiempo] [Función de Tiempo] [Retardo];
}

    [Tiempo]= Refiere al tiempo que durará la transición del elemento definido en propiedades y se definirá en segundos o milisegundos.

    
selector {
    transition:[Propiedad CSS] [Tiempo] [Función de Tiempo] [Retardo];
}

    [Función de Tiempo]= Funciones predefinidas que permiten variar el avance de la transición.

    Tipo de [Función de Tiempo]

    linearLa transición se realiza de manera uniforme.
    easeAcelera al inicio se retarda y se acelera al final.
    ease-in Se retarda al inicio, pero lo repone al final.
    ease-out Acelera al inicio pero se retarda al final.
    ease-in-out Se retarda al inicio, acelera y luego se retarda al final. 

    
selector {
    transition:[Propiedad CSS] [Tiempo] [Función de Tiempo] [Retardo];
}

    [Retardo]= el tiempo que tardará la transición en iniciar y se declara en segundos o milisegundos.

    Ejemplo:

    
div {
  transition: background-color 0.5s ease;
  background-color: red;
}
div:hover {
  background-color: green;
}

    Esta regla indica que nuestro div cambiara de color de rojo a verde en .5 segundos sin retardo.
