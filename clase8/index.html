Propiedad Transform

Aplica una transformación 2D o 3D a un elemento. Esta propiedad permite rotar, escalar, mover, sesgar, etc, elementos.

				
selector{
	transform: método de transformacion(valores de método);
}

Métodos de la propiedadTransform

Esta propiedad es de las más complejas y novedosa de CSS3 y consta de varios métodos de transformación:

Transform: translate()

Mueve al elemento desde su posición original, en función de los parámetros horizontal(eje X) y vertical(eje Y)

				
selector {
	translate:(ejeX,ejeY);
}

Un ejemplo:

Del bloque de imágenes seleccionaremos al primer elemento y le asignaremos los siguientes valores, para cuando el curso se posiciona sobre la imagen:

				
selector?:hover{
-ms-transform: translate(25px,5px); /* IE 9 */
-webkit-transform: translate(25px,5px); /* Chrome, Safari, Opera */
transform: translate(25px,5px);

}

Esta propiedad requiere uso de prefijos para navegador

Transform: rotate()

el elemento gira en sentido de las manecillas del reloj en un grado determinado. Los valores negativos se permiten y gira el elemento hacia la izquierda.

				
selector {
	rotate:(#grados);
}

seleccionaremos la segunda imagen y aplicaremos la propiedad de rotación al:hover

				
selector?:hover{
-ms-transform: rotate(30deg); /* IE 9 */
-webkit-transform: rotate(30deg); /* Chrome, Safari, Opera */
transform: rotate(30deg);
}

Transform: rotate (X°)

El elemento gira sobre el eje de las X (horizontalmente)

				
selector {
	rotateX:(#grados);
}

seleccionaremos la tercera imagen y aplicaremos la propiedad de rotación en X al:hover

				
selector?:hover{
-ms-transform: transform:rotateX(60deg); /* IE 9 */
-webkit-transform:rotateX(60deg); /* Chrome, Safari, Opera */
transform:rotateX(60deg);
}

Transform: rotate(Y°)

Define la rotación de elemento en el eje de las Y

				
selector {
	rotateY:(#grados);
}

En la misma tercera imagen, agregaremos en valor de rotación en Y

				
selector?:hover{
-ms-transform: transform:rotateX(60deg) rotateY(50deg); /* IE 9 */
-webkit-transform:rotateX(60deg) rotateY(50deg); /* Chrome, Safari, Opera */
transform:rotateX(60deg) rotateY(50deg);
}

Transform: rotate 3D(x,y,z,)

Define una rotación simulando un efecto 3D añadiendo el eje Z a la transformación

				
selector {
	transform:rotateX(°) rotateY(°) rotateZ(°);
}

Ahora en la cuarta imagen añadimos la siguiente regla css:

				
selector?:hover{
-ms-transform: transform:rotateX(60deg) rotateY(50deg) rotateZ(60deg); /* IE 9 */
-webkit-transform:rotateX(60deg) rotateY(50deg) rotateZ(60deg); /* Chrome, Safari, Opera */
transform:rotateX(60deg) rotateY(50deg) rotateZ(60deg);
}

Transform: scale()

el elemento aumenta o disminuye el tamaño, dependiendo de los parámetros dados en lo ancho (eje X) y la altura (eje Y):

				
selector {
	scale:(valorX, valorY);
}

En la quinta imagen al :hover

				
selector?:hover{
-ms-transform: scale(2,2); /* IE 9 */
-webkit-transform: scale(2,2); /* Chrome, Safari, Opera */
transform: scale(2,2);
}

Transform: skew()

El elemento gira en un ángulo según los parámetros dados para la (eje X) horizontal y el (eje Y) líneas verticales:

				
selector {
	skew:( valorX°, valorY°);
}

Ahora para la sexta imagen:

				
selector?:hover{
-ms-transform: skew(30deg,20deg); /* IE 9 */
-webkit-transform: skew(30deg,20deg); /* Chrome, Safari, Opera */
transform: skew(30deg,20deg);
}

Transform: skew(X)

Define una transformación de sesgo 2D a lo largo del eje X

				
selector {
	skewX:( valorX°);
}

En la séptima imagen:

				
selector?:hover{
-ms-transform: skewX(40deg); /* IE 9 */
-webkit-transform: skewX(40deg); /* Chrome, Safari, Opera */
transform: skewX(40deg);
}

Transform: skew(Y)

Define una transformación de sesgo 2D a lo largo del eje Y

				
selector {
	skewY:( valorY°);
}

Por la octava imagen:

				
selector?:hover{
-ms-transform: skewY(30deg); /* IE 9 */
-webkit-transform: skewY(30deg); /* Chrome, Safari, Opera */
transform: skewY(30deg);
}

Transform: matrix()

Este método toma seis parámetros, que contiene funciones matemáticas, que le permite: rotar, escalar, mover (trasladar) y sesgar elementos.

				
selector {
	transform:matrix(n1,n2,n3,n4,n5,n6);
}

Por la novena imagen:

				
selector?:hover{
-ms-transform:matrix(1,0.5,-0.5,1,0,0); /* IE 9 */
-webkit-transform:matrix(1,0.5,-0.5,1,0,0); /* Chrome, Safari, Opera */
transform:matrix(1,0.5,-0.5,1,0,0);
}

Propiedad transform-origin

Cuando se aplica una transformación a un objeto, se crea un sistema de coordenadas local. Por defecto, el origen - el punto (0,0) - del sistema de coordenadas local se encuentra en el centro del objeto.

Esta propiedad permite mover el punto de origen del objeto en su eje X y Y.

				
selector {
	transform-origin:(X Y;)
}

Para la ultima imagen:

				
selector?:hover{
	transform: rotate(45deg);
	transform-origin:20% 40%;
	-ms-transform: rotate(45deg); /* IE 9 */
	-ms-transform-origin:20% 40%; /* IE 9 */
	-webkit-transform: rotate(45deg); /* Chrome, Safari, Opera */
	-webkit-transform-origin:20% 40%; /* Chrome, Safari, Opera */
}

Propiedad z-index

La propiedad z-index especifica el orden de apilamiento de un elemento.

Un elemento con mayor orden de apilamiento es siempre frente a un elemento con un orden de apilamiento inferior.

				
selector{
z-index: valor(número positivo o negativo);
}

Agreguemos esta propiedad para lanzar adelante la etiqueta con el nombre del método.

				

.tool-tip {
    background-color: #FF0000;
    display: none;
    z-index: 1;
}

IMPORTANTE:

z-index sólo funciona con objetos que tengan la propiedad position seteada como: relative, fixed o absolute.

Chequemos algunas propiedades que utilizamos:

Propiedad Overflow

Controla el comportamiento de los contenidos que no caben en su elemento contenedor.

visiblemostrará el contenido completo del div.
hiddenocultara el contenido que rebase los bordes del elemento.
scroll añade un scroll para ver el contenido completo del elemento.
auto se adapta y añade la barra de scroll solo si es necesario.

Propiedad Transform translate3d(x,y,z);

Añade un efecto 3d haciendo uso del eje Z.

Propiedad transform-style

Especifica cómo los elementos anidados se prestan en el espacio 3D.

flatEl valor por defecto. Los elementos hijos no conservarán su posición en 3D.
preserve-3dLos elementos anidados o hijos preservarán su posición en 3D

Propiedad transition cubic-bezier()

Por defecto, las funciones de tiempo se definen mediante curva matemática de 6 valores (P1x,P1y,P2x,P2y).

ease-incubic-bezier(0.42, 0.0, 1.0, 1.0)
